# Privacy Policy

TimeDraft is a Chrome extension that accesses Google Calendar data
to generate availability suggestions.

## Data Access
- Reads calendar availability only
- Does not modify calendar events

## Data Storage
- No personal data is stored on external servers
- All processing happens locally in the browser

## Data Sharing
- No data is sold, shared, or transferred to third parties
